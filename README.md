# School-Management-System

**School-Management-System** is a web-based application designed to streamline and automate the administrative and academic processes of a school. The system helps school staff manage students, teachers, classes, subjects, exams, and other school-related activities efficiently.

---

## Features

- **Student Management:** Add, update, and manage student records.  
- **Teacher Management:** Manage teacher profiles, subjects, and assignments.  
- **Class & Subject Management:** Create and organize classes, subjects, and timetables.  
- **Exam & Marks Management:** Conduct exams, record marks, and generate report cards.  
- **Attendance Tracking:** Track student attendance and generate reports.  
- **User Roles:** Admin, Teacher, and Student accounts with role-based access.  
- **Responsive Design:** Works on mobile, tablet, and desktop devices.  
- **Contact & Notifications:** Communicate important notices to staff and students.  

---

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server Environment:** XAMPP (Apache + MySQL)  
- **Optional Libraries:** jQuery, FontAwesome (for icons)  

---

## Purpose of the Project

This project aims to:  

1. Simplify school administrative tasks and record keeping.  
2. Automate processes such as student registration, exam grading, and attendance.  
3. Provide teachers and administrators with a centralized platform to manage school operations.  
4. Serve as a practical project example for students learning PHP, MySQL, and web development.
